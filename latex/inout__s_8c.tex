This function reads .mesh (ASCII) and .meshb (binary) files. If the name contains \char`\"{}.\+mesh\char`\"{} the file will be read as an ASCII file and if the name contains .meshb it be read as a binary file. If the file contains neither of these strings the function will first try to open \char`\"{}\mbox{[}filename\mbox{]}.\+meshb\char`\"{} and if this fails it will try \char`\"{}\mbox{[}filename\mbox{]}.\+mesh\char`\"{}.
