The Doxygen documentation of the last release of Mmg is available on \+: \href{https://mmgtools.github.io/}{\texttt{ https\+://mmgtools.\+github.\+io/}}\hypertarget{md_doc_doxygen_BUILD_DOXYGEN_DOC_autotoc_md109}{}\doxysection{Build locally your own html documentation}\label{md_doc_doxygen_BUILD_DOXYGEN_DOC_autotoc_md109}
You can build the Mmg documentation using the {\ttfamily make doc} command generated by CMake.

For CMake begginers you can follow the next steps\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{cd mmg}
\DoxyCodeLine{mkdir build}
\DoxyCodeLine{cd build}
\DoxyCodeLine{cmake ..}
\DoxyCodeLine{make doc}

\end{DoxyCode}


By default the Doxygen output is builded by the under the {\ttfamily doc} sub directory of the build directory (so in {\ttfamily build/doc} in the previous example). You can configure the {\ttfamily MMG\+\_\+\+DOC\+\_\+\+DIR} CMake variable to change the Doxygen output directory.
