Calculate the list of all the triangles intersected by edge (ia,ib), starting from kdep = one triangle in the ball of ia; {\itshape list} starts at index 0 (and ends at index lon-\/1). It stores 3$\ast$k + iare, where {\itshape k} is an intersected tria and {\itshape iare} is an intersected edge. prod1, prod2 and prod3 allow to localize the edge regarding the tria\+:
\begin{DoxyItemize}
\item if all are \texorpdfstring{$>$}{>}0, the edge doesn\textquotesingle{}t cross the tria (same sign for all areas)
\item if 2 are negative and 1 is positive, the edge intersect 2 triangle edges
\item if 2 are 0, the edge pass through 1 of the tria vertices
\item if 3 are 0, the edge coincide with 1 of the tria edges
\end{DoxyItemize}
