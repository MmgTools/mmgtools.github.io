To call the {\bfseries{mmg2d}} library, you must\+: ~\newline

\begin{DoxyEnumerate}
\item build mesh and sol at MMG5 format;
\item call the MMG2D library ({\bfseries{MMG2\+D\+\_\+mmg2dmesh}} function);
\item get the final mesh and sol.
\end{DoxyEnumerate}

We read mesh and solution files using the {\bfseries{MMG2\+D\+\_\+load\+Mesh}} and {\bfseries{MMG2\+D\+\_\+load\+Sol}} functions.

We ask for a maximal edge size of 0.\+1 using the {\bfseries{MMG2\+D\+\_\+\+Set\+\_\+dparameter}} function. We increase the verbosity level using the {\bfseries{MMG2\+D\+\_\+\+Set\+\_\+iparameter}} function.
