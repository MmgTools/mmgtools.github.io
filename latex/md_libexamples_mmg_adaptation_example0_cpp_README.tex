To call the {\bfseries{mmg}} library, you must\+: ~\newline

\begin{DoxyEnumerate}
\item build mesh and sol at MMG2D, MMG3D or MMGS format;
\item call the MMG2D, MMGS or MMG3D library;
\item get the final mesh and sol.
\end{DoxyEnumerate}

We read mesh and solution files using the {\bfseries{MMG\texorpdfstring{$<$}{<}2D/\+S/3D\texorpdfstring{$>$}{>}\+\_\+load\+Mesh}} and {\bfseries{MMG\texorpdfstring{$<$}{<}2D/\+S/3D\texorpdfstring{$>$}{>}\+\_\+load\+Sol}} functions. Results are saved using {\bfseries{MMG\texorpdfstring{$<$}{<}2D/\+S/3D\texorpdfstring{$>$}{>}\+\_\+save\+Mesh}} and {\bfseries{MMG\texorpdfstring{$<$}{<}2D/\+S/3D\texorpdfstring{$>$}{>}\+\_\+save\+Sol}} functions.\hypertarget{md_libexamples_mmg_adaptation_example0_cpp_README_autotoc_md37}{}\doxysection{II/ Compilation}\label{md_libexamples_mmg_adaptation_example0_cpp_README_autotoc_md37}

\begin{DoxyEnumerate}
\item Build and install the {\bfseries{mmg}} shared and static library. We suppose in the following that you have installed the {\bfseries{mmg}} library in the $\ast$$\ast$\+\_\+\$\+CMAKE\+\_\+\+INSTALL\+\_\+\+PREFIX\+\_\+$\ast$$\ast$ directory (see the \href{https://github.com/MmgTools/Mmg/wiki/Setup-guide\#iii-installation}{\texttt{ installation}} section of the setup guide);
\item compile the \mbox{\hyperlink{main_8cpp}{main.\+cpp}} file specifying\+:
\begin{DoxyItemize}
\item the {\bfseries{mmg}} include directory with the {\bfseries{-\/I}} option;
\item the {\bfseries{mmg}} library location with the {\bfseries{-\/L}} option;
\item the {\bfseries{mmg}} library name with the {\bfseries{-\/l}} option;
\item for the static library you must also link the executable with, if used for the {\bfseries{mmg}} library compilation, the scotch and scotcherr libraries and with the math library;
\item with the shared library, you must add the $\ast$$\ast$$\ast$\+\_\+\$\+CMAKE\+\_\+\+INSTALL\+\_\+\+PREFIX\+\_\+$\ast$$\ast$ directory to your {\bfseries{LD\+\_\+\+LIBRARY\+\_\+\+PATH}}.
\end{DoxyItemize}
\end{DoxyEnumerate}
