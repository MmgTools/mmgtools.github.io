Count the number of entities of eache type (points, triangles...) in the mesh as well as the number of node data (solutions). If a data field name contains the \char`\"{}medit\+:ref\char`\"{} string, it will be used as entity reference instead of a solution. If node references are detected, {\itshape pt\+Medit\+Ref} is incermented. If cell references are detected, {\itshape elt\+Medit\+Ref} is incremented.

\begin{DoxyRemark}{Remarks}
Note that Mmg supports only 1 reference by point and 1 reference per cell. 
\end{DoxyRemark}
