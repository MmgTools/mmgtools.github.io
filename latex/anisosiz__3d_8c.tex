Use metric intersection to gradate the anisotropic metric on point P, given the metric extended from point Q on the edge PQ.
\begin{DoxyEnumerate}
\item On singular (CRN, REQ, NOM) points, the metric on P is isotropic (as in MMG5\+\_\+defmetsin) and should remain isotropic.
\item On non-\/singular ridge points, the metric on P should remain aligned with the ridge directions (thus it is not possible to apply metric intersection, sizes are truncated instead).
\item On regular boundary points, the metric can be anisotropic on the tangent plane, but it should remain aligned to the normal direction (thus, intersection is used only in the tangent plane and sizes are truncated in the normal direction).
\item On interior volume points, 3D metric intersection can be used. 
\end{DoxyEnumerate}
