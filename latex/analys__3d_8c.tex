preprocessing stage\+: mesh analysis.

At the end of this function\+:
\begin{DoxyEnumerate}
\item triangles have been deleted and stored in xtetra;
\item Boundary point have been analyzed and associated to xpoints (if needed). If computed tangent is stored in ppt-\/\texorpdfstring{$>$}{>}n and normals in pxp-\/\texorpdfstring{$>$}{>}n1 and pxp-\/\texorpdfstring{$>$}{>}n2. Following list summerizes computed (and not computed) data depending on point type\+:
\begin{DoxyItemize}
\item Corner point (MG\+\_\+\+CRN)\+: no computation (and no xpoint);
\item Required points have different treatment depending if they are along a \textquotesingle{}regular\textquotesingle{} surf point, along an internal or external non-\/manifold edge, along a non connex mesh part(so they may have a xpoint, a tangent and a normal or not) but their normal is not used during remeshing. \item Reference point (MG\+\_\+\+REF)\+: xpoint, tangent, 1 normal;
\item Ridge point (MG\+\_\+\+GEO)\+: xp, tangent,2 normals;
\item Non-\/manifold point (MG\+\_\+\+NOM)\+: xp, tangent, no normal if internal, 1 normal if external
\item Open boundary points (MG\+\_\+\+OPNBDY) are treated as nm points. 
\end{DoxyItemize}

Define continuous geometric support at non manifold vertices, using volume information.

Following list summerizes computed data depending on point type\+:
\begin{DoxyItemize}
\item Non-\/manifold external point along connex mesh part (MG\+\_\+\+NOM)\+: xp, tangent, normal n1, nnor=0
\item Non-\/manifold external point along edge or point connected mesh part (MG\+\_\+\+NOM)\+: no xp,no tangent,no normal,nnor=0
\item Non-\/manifold not required internal point (MG\+\_\+\+NOM+\+MG\+\_\+\+REQ)\+: xp, tangent, no normal, nnor=1.
\item Non-\/manifold required internal point (MG\+\_\+\+NOM+\+MG\+\_\+\+REQ)\+: no xp, no tangent, no normal, nnor=1.
\item Non-\/manifold open-\/boundary (MG\+\_\+\+OPNBDY) edges are treated like others\+:
\begin{DoxyItemize}
\item if we are along a \char`\"{}truly\char`\"{} open boundary, it is an internal nm edge so we don\textquotesingle{}t have any normal;
\item if the edge is marked as open while being only non-\/manifold, we may compute a normal (along an external boundary) or not (along an internal one).
\end{DoxyItemize}
\item We don\textquotesingle{}t compute anything along corner points. 
\end{DoxyItemize}

Compute normals at C1 vertices, for C0\+: tangents.

Following list summerizes computed (and not computed) data depending on point type\+:
\begin{DoxyItemize}
\item Corner point (MG\+\_\+\+CRN)\+: nothing (and no xpoint);
\item Reference point (MG\+\_\+\+REF)\+: xp, tangent (ppt-\/\texorpdfstring{$>$}{>}n), 1 normal (pxp-\/\texorpdfstring{$>$}{>}n1);
\item Ridge point (MG\+\_\+\+GEO)\+: xp, tangent,2 normals (pxp-\/\texorpdfstring{$>$}{>}n1 and n2);
\item Non-\/manifold point (MG\+\_\+\+NOM) are not filled here but in {\itshape nmgeom} function
\item Open boundary points (MG\+\_\+\+OPNBDY) are non-\/manifold so they are filled in nmgeom too.
